## Step1. 花Xの周りの設定

花Xの周りに木をおく。
花の周りの配置方法のうち、以下のパターンから1つをランダムに選ぶようにする
`*`は木をおいていけない箇所である。この位置にすでに`T`が立っている場合や、外壁になっているような場合は採用しない

### パターン1

```
.#.
**#
#X#
.#.
```

### パターン2

```
.#.
#**
#X#
.#.
```

### パターン3

```
.#.
#X#
**#
.#.
```

### パターン4

```
.#.
#X#
#**
.#.
```

### パターン5

```
.*#.
#*X#
.##.
```

### パターン6

```
.##.
#*X#
.*#.
```

### パターン7

```
.#*.
#X*#
.##.
```

### パターン8

```
.##.
#X*#
.#*.
```
